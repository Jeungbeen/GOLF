# GOLF by Jeungbeen
*A drag and drop ~~heavily unoptimized~~ game by me*

### How to play:
*1. Get three different items each named "Driver" / "Hybrid" / "Wedge", "Putter" and "Flag"*

*2. [ALT] + [LClick] with the flag equipped to place a hole*

*3. [ALT] + [LClick] with any of the golf clubs equipped to start the game, you will not be able to place any more holes*

*4. [ALT] + [RClick] with any of the golf clubs equipped to toggle guidelines*


### Commands:
*Type /golf help for all available commands*

*SuperTeleport.lua is optional, however /golf tp needs it to function*
