# How to play:
1. Get three different items each named "Driver" / "Hybrid" / "Wedge", "Putter" and "Flag"
2. Click Left Alt and Left Mouse with the flag equipped to place a hole
3. Click Left Alt and Left Mouse with any of the golf clubs equipped to start the game, you will not be able to place any more holes
4. Click Left Alt and Right Mouse with any of the golf clubs equipped to toggle guidelines


# Commands:
Type /golf help for all available commands
